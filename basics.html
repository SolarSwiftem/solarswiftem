Lesson 1:

Objective:

In this lesson you will learn how to print out your very first line of code! This is useful when you require the computer to output some data. You will also learn how to comment on your code which is helpful to you or anyone else reading the code to understand what is going on without having to read each line of code.

Demonstration:

Videos will be added after/during reading week

The code file from the video should be attached. Write this before the video is recorded and use it to guide the video

Exercises:

  • Write a program that prints you address on separate lines
  • Comment out your address
