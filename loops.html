Lesson 2: For Loops

Objective:

In this lesson you will learn about for loops. For loops are the most commonly used loop. In comparison to other loops, for loops explicitly make use of a counter to progress towards a termination point.

For loops take the form:

for initialization; condition; increment { }

Where initialization is where the counter is initially set, condition is the condition that is tested at the end of the block of code within the loop and increment is where the change to counter is stated.

e.g.

for var i=0; i<3; i++ {
// Code to run goes here
}

In the above example, the counter i is set to 0 and the block of code within the loop will run. After the first time the code runs, i is incremented.

Lesson 3: Repeat-while loops

Objective:

In this lesson you will learn how to use repeat-while loops. Repeat-while loops operate in a very similar manner to while loops. The main separator is that the block of code within the loop will execute at least once before the condition is tested to see if the loop will continue to run.
