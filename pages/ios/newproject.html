Lesson 1: Starting a new project and running

Making apps can sound hard but this lesson will show you how easy it is to start out with something basic. You will see how quick and easy it is to get an app onto a device.

Goal:

In this lesson, you will learn how to create a new project for an app. The app won’t be that fancy but you will learn how to run it on a simulator as well as on a real iPhone/iPad.
