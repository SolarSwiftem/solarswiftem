Lesson 6: Functions

Functions are a big part of programming. They are used to run a chunk of code when we need it. If we use the same code a lot in different parts of the program it can be useful to put them in a function so that we don’t have to repeat the code all over the place.

Goal

In this lesson you will learn how to create functions and how to call them in other pieces of code. This includes functions with parameters and return values.

Exercises

  • Write a program with a function that prints your name. Use that function to print your name 20 times.
  • Write a program with a function that takes 3 numbers and adds them together. Use that function to add the numbers 123, 231 and 321 and print the answer returned from that function.
  • Write a program with a function that returns an array with all the numbers from 1-100. Write another function that takes an array of numbers and returns an array with all the values from the first array multiplied by 4. Print the result of calling the second function on the value returned from the first function.
