Lesson 8: A Whole App

It is finally time to put all of your knowledge together into one full app! You will be making an app from start to finish so get ready.

Goal:

In this lesson, you will create a fully working app from start to finish. This will require all the skills that you have learned in the previous lessons.

Demonstration:

Videos will be added after/during reading week

The code file from the video should be attached. Write this before the video is recorded and use it to guide the video

Exercises:

(Aim to write 3-5 exercise questions of varying difficulty. Make sure they highlight the different types of tricks/challenges that your concept may possess.)

  • Look at the example apps for this lesson. Some of them have different things missing. Can you figure out what is wrong with them and fix them?
  • Make an app which tells somebody about yourself. The app should have different pages for different things about you. Try using tables and images to describe what you like.
